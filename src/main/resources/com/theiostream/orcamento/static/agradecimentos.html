Sobre

Este site foi criado entre 2014 e 2015. Em temporada eleitoral, era comum ouvir dados sobre investimentos que deixaram de ser feitos tanto em debates quanto na mídia (seja jornais ou ONGs dedicadas à análise de gastos públicos), sendo difícil confirmar a veracidade deles dados os meios existentes.

Até então, o meio estabelecido para a navegação pelo Orçamento constantemente citado como fonte era o Siga Brasil, mantido pelo Senado Federal. A navegação pelos dados, contudo, é difícil, lenta, e não mostra qualquer visualização gráfica. Houve uma tentativa por parte da organização não-governamental INESC para melhorar a visualização destes dados, denominada Orçamento ao Seu Alcance (que só descobri quanto já tinha feito bastante progresso neste site). Contudo, o nível de navegação proporcionado era muito superficial e não mostrava todos os dados a fundo.

Em 2013, inspirado na Lei de Acesso à Informação, a Secretaria do Orçamento Federal, do Ministério do Planejamento, criou um índice em formato RDF (excelente para processamento deste tipo de dados relacionais) das despesas de acordo com a base de dados SIOP, que possui um índice das despesas orçamentárias, cuja especificação encontra-se bem documentada. Entretanto, não havia até então qualquer interface que fizesse uso desses dados para mostrá-los.

Assim, foi desenvolvido este website para tentar democratizar o acesso aos dados de despesas públicas, mostrando-os de maneira acessível e fácil de navegar, preservando contudo a profundidade dos dados. Fiel a este espírito, todo o conteúdo é colocado sob domínio público e o código é software livre.

Verificação de Valores

Este site está em estágio beta. Por favor confira dados da Lei Orçamentária Anual para verificar sua validade.

Este website é construído sobre a base de dados SIOP do Ministério do Planejamento, que é bem diferente da base de dados SIAFI do Tesouro Nacional, utilizada pelo Portal da Transparência. Deve ser esperado que valores para o mesmo recurso mostrado neste site seja diverso dos valores do Portal. A SIOP, por sua vez, é uma das fontes que alimentam o Siga Brasil, portanto, em boa parte dos casos, os valores devem ser os mesmos.
